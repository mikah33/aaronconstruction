# Parenteau Property Development Website

Professional property development website for Parenteau Property Development, serving Rhode Island.

## Features

- **Responsive Design**: Mobile-first, fully responsive across all devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Service Showcase**: Comprehensive display of all development services
- **Contact Integration**: Easy-to-use contact form and click-to-call functionality
- **Social Media Links**: Integrated Facebook and Instagram links
- **SEO Optimized**: Proper meta tags and semantic HTML
- **Fast Loading**: Optimized assets and code

## Technologies Used

- HTML5
- CSS3 (Custom + Bootstrap 5)
- JavaScript (Vanilla)
- Bootstrap 5.3
- AOS (Animate On Scroll)

## Setup

1. Download all files
2. Add your project images to the `/images` folder
3. Open `index.html` in a web browser
4. Customize content as needed

## Image Requirements

Add the following images to the `/images` folder:
- `hero-bg.jpg` - Hero section background
- `about-placeholder.jpg` - About section image
- `project-1.jpg` through `project-6.jpg` - Gallery images

Recommended dimensions:
- Hero background: 1920x1080px
- About image: 800x600px
- Gallery images: 600x400px

## Customization

To customize the website:
1. Edit company information in `index.html`
2. Modify colors in `css/style.css` (root variables)
3. Add/remove services in the services section
4. Update contact form endpoint in `js/script.js`
